# PRD (Product Requirement Document) - RAV4 Road Trip

## 1. Overview
미국의 아름다운 들판과 고속도로를 배경으로 Toyota RAV4 차량을 운전하며 장애물을 피하는 웹 기반 플래시 스타일 미니게임.

## 2. Target Audience & Platform
- HTML5 호환 웹 브라우저 (데스크톱 및 모바일)

## 3. Core Features
- **3차선 종방향 레이싱**: 종방향(세로형) 도로에서 자동차가 진행하며 상단에서 하단으로 장애물이 스크롤됨.
- **조작 방식**: 
  - 화면 하단의 좌/우 화살표 버튼 클릭/터치
  - 키보드 방향키 및 A/D 키 지원
  - 부드러운 차선 이동 애니메이션 (Lerp 기반)
- **장애물 유형**:
  - 돌(Rock), 고속도로 주변의 나무 등
  - 귀여운 동물 (강아지 🐶): 강아지를 충돌하면 강아지가 다치는 대신 플레이어가 급제동(Warning 경고, 속도 급감, HP 소폭 차감)하여 안전하게 피해가는 연출.
  - 미국식 교통 차량: 미국식 대형 트럭(Semi-Truck) 및 일반 세단 차량.
- **배경 연출**: 미국의 들판, 나무, 빨간색 농장(Barn), 풍차(Windmill)가 지나가는 패럴랙스(Parallax) 스크롤링 효과.

## 4. UI/UX
- 프리미엄 글래스모피즘(Glassmorphism) 기반 시작/게임오버 오버레이 카드.
- 상단 HUD: 주행 거리(Score), 차량 HP 상태바, 현재 속도(Speed).
- 다크 모드 기반의 세련된 아케이드 감성의 레이아웃.
